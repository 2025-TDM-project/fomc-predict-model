📊 FOMC タカ派・ハト派 予測モデル
本プロジェクトは、FOMCの声明文、債券レポート、ニュース記事など、多様なテキストデータを活用し、米国金融政策の**タカ派（Hawkish）– ハト派（Dovish）**的な傾向を定量化するテキスト分析パイプラインです。

トピックモデリングとドメイン辞書を組み合わせることで、文書単位の連続型スコア（-1 ～ 1）を算出することを目標としています。

プロジェクト概要
本プロジェクトの目的は、金融政策に関連するテキストから政策スタンスを抽出し、それを確率ベースのスコアに変換することで、政策トーンの変化を定量的に分析することにあります。


分析パイプライン

1. データ収集
FOMC 声明文および議事録
債券市場レポート（ウェブスクレイピング）
ニュースデータ（New York Times API）

2. テキスト前処理
小文字変換および正規化
特殊文字およびストップワード（不英語）の除去
文書の長さ制限および精緻化

3. 補題化（Lemmatization）およびコロケーションの適用
補題化による言語的な正規化
Bigram / Trigram / Fourgram コロケーションの生成
主要な経済表現を単一トークンとして置換
（例: interest_rate, rate_hike, price_stability）

4. トピックモデリング
BERTopic を活用したトピックモデリングの実行
コロケーションを適用したテキストを入力データとして使用
各文書へのトピックID割り当て

5. タカ派・ハト派辞書の構築
コロケーション統計を活用したシード単語（Seed Words）の自動抽出
ドメイン知識を反映した手動によるシード補正
最終的な辞書の構成
タカ派（Hawkish）辞書
ハト派（Dovish）辞書

6. 感情スコアの算出
トピック単位でのタカ派・ハト派傾向の計算
文書単位の確率ベーススコア（-1 ～ 1）の算出

閾値（Threshold）に基づいた文書のラベル付け
Hawkish / Dovish / Neutral

7. TF-IDF（オプション）
初期実験段階では適用したが、最終分析段階ではトピックベースの意味情報を優先するため除外。

算出結果（アウトプット）
·文書単位のタカ派・ハト派スコア
·文書ラベル（Hawkish / Dovish / Neutral）
·トピック別サマリーテーブル
·主要単語（Top Words）
·平均感情スコア
·文書数
