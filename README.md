📊 FOMC 매파·비둘기파 예측 모델

본 프로젝트는 FOMC 문서, 채권 리포트, 뉴스 기사 등 다양한 텍스트 데이터를 활용하여
미국 통화정책의 매파(hawkish)–비둘기파(dovish) 성향을 정량화하는 텍스트 분석 파이프라인이다.

토픽 모델링과 도메인 사전을 결합하여 문서 단위의 연속형 점수 (-1 ~ 1) 산출하는 것을 목표로 한다.

⸻

프로젝트 개요

본 프로젝트의 목적은 통화정책 관련 텍스트로부터 정책 기조를 추출하고,
이를 확률 기반 점수로 변환하여 정책 톤의 변화를 정량적으로 분석하는 것이다.

⸻

분석 파이프라인

1. 데이터 수집
	•	FOMC 성명문 및 의사록
	•	채권 시장 리포트 (웹 크롤링)
	•	뉴스 데이터 (New York Times API)

⸻

2. 텍스트 전처리
	•	소문자 변환 및 정규화
	•	특수문자 및 불용어 제거
	•	문서 길이 제한 및 정제

⸻

3. 표제어 추출 및 Collocation 적용
	•	표제어 추출(Lemmatization)을 통한 언어적 정규화
	•	Bigram / Trigram / Fourgram collocation 생성
	•	주요 경제 표현을 단일 토큰으로 치환
	•	예: interest_rate, rate_hike, price_stability

⸻

4. 토픽 모델링
	•	BERTopic 기반 토픽 모델링 수행
	•	Collocation이 적용된 텍스트를 입력으로 사용
	•	각 문서에 토픽 ID 할당

⸻

5. 매파·비둘기파 사전 구축
	•	Collocation 통계를 활용한 자동 시드 단어 추출
	•	도메인 지식을 반영한 수동 시드 보정
	•	최종 사전 구성
	•	매파(hawkish) 사전
	•	비둘기파(dovish) 사전

⸻

6. 감성 점수 산출
	•	토픽 단위 매파·비둘기파 성향 계산
	•	문서 단위 확률 기반 점수(-1 ~ 1) 산출
	•	임계값 기반 문서 라벨링
	•	hawkish
	•	dovish
	•	neutral

⸻

7. TF-IDF (선택적)
	•	초기 실험 단계에서 적용하였으나 
	•	최종 분석 단계에서는 토픽 기반 의미 정보를 우선하여 제외

⸻

산출 결과
	•	문서 단위 매파·비둘기파 점수
	•	문서 라벨(hawkish / dovish / neutral)
	•	토픽별 요약 테이블
	•	주요 단어(top words)
	•	평균 감성 점수
	•	문서 수
